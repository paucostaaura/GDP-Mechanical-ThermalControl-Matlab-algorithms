# GDP-Mechanical-ThermalControl-Matlab-algorithms
MATLAB screening and sizing tools for spacecraft thermal control and mechanical analysis developed for the GDP mission design project.

# NEO Pathfinder – Mechanical and Thermal MATLAB Codes

This repository contains the MATLAB scripts developed to support the mechanical and thermal design of the NEO Pathfinder spacecraft during the Phase 0/A study.

## Contents
- Geometry trade-off analysis
- Solar array structural sizing
- Internal layout generation
- Supporting design and verification tools
